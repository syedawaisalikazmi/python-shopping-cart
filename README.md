# 🛒 Python Shopping Cart System

## 📌 Description

This is a simple command-line shopping cart system built in Python.
Users can add food items with prices, view their cart, and get the total bill.

## 🚀 Features

* Add multiple food items
* Store data using lists
* Display cart with item-price pairing (using zip)
* Calculate total bill
* Exit anytime using 'q'

## 🛠️ Tech Used

* Python

## ▶️ How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/python-shopping-cart.git
   ```
2. Open the folder
3. Run the program:

   ```bash
   python shopping_cart.py
   ```

Enter food name: Burger
